# Task Manager
CLI - приложение, для контроля над задачами, с возможностью удаления, обновления, удаления задач. Сохранение задач на файле

Структура проекта TaskManager

- TaskManager
- ├── src
- │   ├── main.cpp
- │   ├── Task.cpp
- │   └── TaskManager.cpp
- ├── include
- │   ├── Task.h
- │   └── TaskManager.h
- ├── data
- │   └── tasks.json  (или tasks.txt)
- ├── CMakeLists.txt  (если используете CMake)
- └── README.md

Описание структуры
- **src** - Директория для исходного кода.
	- main.cpp: Главный файл, содержащий точку входа в программу. Здесь будет реализован интерфейс командной строки для взаимодействия с пользователем.
	- Task.cpp: Реализация методов класса Task.
	- TaskManager.cpp: Реализация методов класса TaskManager, который будет управлять задачами.
 - **include** - Директория для заголовочных файлов.
      - Task.h: Заголовочный файл для класса Task, который будет представлять отдельную задачу.
      - TaskManager.h: Заголовочный файл для класса TaskManager, который будет управлять коллекцией задач.
  - **data** -  Директория для хранения данных.
      - tasks.json или tasks.txt: Файл, в котором будут храниться задачи. Вы можете выбрать формат JSON или текстовый, в зависимости от ваших предпочтений.
-  **README.md** - Файл с описанием проекта, инструкциями по установке и использованию.
